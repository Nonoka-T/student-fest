# Student Fest Encoder/Decoder

This project is a simple, educational encoder/decoder built for **Student Fest**.  
It demonstrates two classic techniques: **Caesar Cipher** and **Base64 Encoding**.

---

## 🚀 Features
- **Caesar Cipher**  
  - Shift letters by a fixed amount (default: 3).  
  - Supports both encoding and decoding.  

- **Base64 Encoding**  
  - Convert text into Base64 format.  
  - Decode Base64 back into plain text.  

- **Menu-driven CLI**  
  - Easy to use interactive interface (`main.py`).  
  - Choose between Caesar or Base64 operations.
