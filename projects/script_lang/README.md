# 📂 Project: Student-Fest Script Language
**Difficulty**: Higher-Intermediate  
**Topics**: Compiler Logic Creativity and Understanding of Systems Programming

## 📝 Project Description

create a Simple Compiler or a Interpreter in any language of your choice and the Syntax is also of your choice 
You can make it like Python and Java or like you dont compile you Just Run it on Your Code 

## 🎯 Core Features

1.  **Command-Line Arguments**: it should have Commandline Arguments for Comiplation 

2.  **The Syntax of the Language**:  It has to be simple
3.  **Categorization**: You can also make You Inter preter Run your Own Byte code files

4.  **Custom Code File EXtenssions**: You can make Your own Language have its own File Extension! 
5.  **Good Error Promts**: Print the errors in the CLI terminal

## 🧪 How to Contribute

1.  Create a new directory for your solution under `projects/script_lang/solution`.
2.  Develop your CLI tool within that directory.
3.  Update this `README.md` with clear instructions on how to install and run your script.
4.  Open a pull request with your contribution.

This project is a practical way to learn about systems Programming and building useful developer tools. Good luck! 🚀
